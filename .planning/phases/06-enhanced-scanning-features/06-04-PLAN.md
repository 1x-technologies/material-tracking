  <action>
    **1. Add `scan.requestSignatureLink` mutation** (protectedProcedure — admin or driver):
    - Input schema: `z.object({ shipmentId: z.string().min(1), pieceId: z.string().min(1) })`
    - Implementation:
      - Generate a cryptographically random token: `crypto.randomBytes(32).toString("hex")` (per Pitfall 5)
      - Create Firestore document at `signatureRequests/{token}` with:
        ```typescript
        {
          shipmentId: input.shipmentId,
          pieceId: input.pieceId,
          createdBy: ctx.user.uid,
          createdAt: FieldValue.serverTimestamp(),
          expiresAt: new Date(Date.now() + 7 * 24 * 60 * 60 * 1000), // 7 day expiry
          consumedAt: null,
        }
        ```
      - Return `{ token, url: `/sign/${token}` }` — the frontend constructs the full URL using `window.location.origin`

    **2. Add `scan.submitSignatureByToken` mutation** (publicProcedure — NO auth required):
    - Input schema: `z.object({ token: z.string().min(1), signatureData: z.string().min(1) })` — signatureData is base64 PNG data URL
    - Implementation:
      - Read `signatureRequests/{token}` from Firestore
      - Validate: doc exists, `consumedAt` is null, `expiresAt > now`
      - Throw `NOT_FOUND` if token invalid/expired/consumed
      - Decode base64 data URL to Buffer: strip `data:image/png;base64,` prefix, `Buffer.from(base64, "base64")`
      - Upload via Admin SDK: `storage.bucket().file(`signatures/${shipmentId}/${pieceId}/${Date.now()}.png`).save(buffer, { contentType: "image/png", public: false })`
      - Get download URL: use `file.getSignedUrl({ action: "read", expires: "2099-01-01" })` for a long-lived URL, or use `getDownloadURL` pattern. **Simplest approach:** `storage.bucket().file(path).getSignedUrl({ action: "read", expires: "03-01-2099" })` to get a signed read URL.
      - Update piece: `db.doc(`shipments/${shipmentId}/pieces/${pieceId}`).update({ deliverySignatureUrl: url, updatedAt: FieldValue.serverTimestamp() })`
      - Mark token consumed: `db.doc(`signatureRequests/${token}`).update({ consumedAt: FieldValue.serverTimestamp() })`
      - Return `{ success: true }`

    **3. Import `crypto` from Node.js** at top of file:
    - `import crypto from "node:crypto"`

    **4. Import `publicProcedure`** from `../trpc` for the unauthenticated endpoint.
