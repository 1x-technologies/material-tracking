Purpose: This is the core scan workflow engine. Every scan in the system flows through this mutation.

    **`validateTransition(currentStatus, action)`:**
    - Import `PieceStatus` and `ScanAction` types from `@material-tracking/shared`
    - Define transition map: `in_transit` requires current `created`; `delivered` requires current `in_transit`; `completed` requires current `delivered` (per D-04)
    - Return `{ valid: true, newStatus }` on success
    - Return `{ valid: false, error: "INVALID_TRANSITION: cannot go from {current} to {action}" }` on invalid skip
    - Return `{ valid: false, error: "ALREADY_AT_STATUS: piece is already {current}" }` if current matches or is ahead of action

    **`deriveShipmentStatus(pieceStatuses[], currentShipmentStatus)`:**
    - Import `ShipmentStatus` type from `@material-tracking/shared`
    - If `currentShipmentStatus === "cancelled"` → return `{ status: "cancelled" }`
    - Count pieces by status
    - If all `completed` → `{ status: "completed" }`
    - If all `delivered` or `completed` but not all `completed` → `{ status: "delivered" }`
    - If any `delivered` but not all at least `delivered` → `{ status: "partially_delivered", deliveredCount: N, pieceCount: total }`
    - If any `in_transit` → `{ status: "in_transit" }`
    - Else → `{ status: "created" }`

    1. Create `apps/api/src/routers/scan.ts`:
    - Import `processScanSchema` from `@material-tracking/shared`
    - Import `protectedProcedure` from `../middleware/auth` (D-05: any authenticated user)
    - Import `router` from `../trpc`
    - Import `db` from `../lib/firebase`
    - Import `validateTransition`, `deriveShipmentStatus` from `../lib/shipment-status`
    - Import `FieldValue`, `Timestamp` from `firebase-admin/firestore`
    - Import `TRPCError` from `@trpc/server`

    - Export `scanRouter = router({ process: ... })`:
      - `process`: `protectedProcedure.input(processScanSchema).mutation(async ({ ctx, input }))`
      - Implementation:
        a. `const qr = input.qrCode.trim()`
        b. Collection group lookup outside transaction (read-only, for pieceRef):
           `db.collectionGroup("pieces").where("qrCode", "==", qr).limit(2).get()`
        c. If `snap.empty` → throw `TRPCError({ code: "NOT_FOUND", message: "UNKNOWN_QR" })`
        d. If `snap.size > 1` → throw `TRPCError({ code: "INTERNAL_SERVER_ERROR", message: "DUPLICATE_QR" })`
        e. `const pieceRef = snap.docs[0].ref`
        f. `const shipmentRef = pieceRef.parent.parent!`
        g. Run `db.runTransaction(async (tx) => { ... })`:
           - `const [pieceSnap, shipmentSnap] = await Promise.all([tx.get(pieceRef), tx.get(shipmentRef)])`
           - If `!pieceSnap.exists || !shipmentSnap.exists` → throw NOT_FOUND
           - `const pieceData = pieceSnap.data()!`
           - `const shipmentData = shipmentSnap.data()!`
           - If `shipmentData.status === "cancelled"` → throw `TRPCError({ code: "PRECONDITION_FAILED", message: "SHIPMENT_CANCELLED" })`
           - `const result = validateTransition(pieceData.status, input.action)`
           - If `!result.valid` → throw `TRPCError({ code: "CONFLICT", message: result.error })`
           - Build event: `{ action: input.action, timestamp: Timestamp.now(), userId: ctx.user.uid, userName: ctx.user.name ?? ctx.user.email ?? "" }`
           - Build piece update: `{ status: result.newStatus, events: FieldValue.arrayUnion(event), updatedAt: FieldValue.serverTimestamp() }`
           - If `input.action === "delivered"` → add `deliveredAt: FieldValue.serverTimestamp()` to update
           - If `input.action === "completed"` → add `completedAt: FieldValue.serverTimestamp()` to update
           - `tx.update(pieceRef, pieceUpdate)`
           - Read all sibling pieces: `const allPiecesSnap = await tx.get(shipmentRef.collection("pieces"))`
           - Build statuses array: map each piece to its status, but use `result.newStatus` for the current piece (since tx.update hasn't flushed yet)
           - `const derived = deriveShipmentStatus(allStatuses, shipmentData.status)`
           - Build shipment update: `{ status: derived.status, updatedAt: FieldValue.serverTimestamp() }`
           - If `derived.deliveredCount !== undefined` → add `deliveredPieceCount: derived.deliveredCount` to shipment update
           - `tx.update(shipmentRef, shipmentUpdate)`
           - Return `{ pieceId: pieceRef.id, shipmentId: shipmentRef.id, newStatus: result.newStatus, shipmentNumber: shipmentData.shipmentNumber, pieceNumber: pieceData.pieceNumber }`

    2. Update `apps/api/src/router.ts`:
    - Add `import { scanRouter } from "./routers/scan"`
    - Add `scan: scanRouter` to the `appRouter` router object
