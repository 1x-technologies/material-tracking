**Step 2: Create `apps/functions/src/lib/slack.ts`**

Three exported functions following the lazy-init singleton pattern from `firebase.ts`:

1. `getSlackClient()` -- Lazy-initialized WebClient singleton. Calls `getSecret("slack-bot-token")` on first use. Caches the WebClient instance in a module-level variable.
   ```typescript
   import { WebClient } from "@slack/web-api";
   import { getSecret } from "./firebase";
   ```

2. `lookupSlackUser(email: string): Promise<string | null>` -- Calls `client.users.lookupByEmail({ email })`. Returns `result.user?.id ?? null`. On error, if `error.data?.error === "users_not_found"`, log `console.warn("[slack] User not found for email: ${email}")`. For any other error, log `console.error("[slack] lookupByEmail failed for ${email}:", error)`. Always returns null on error (per D-15).

3. `sendSlackDM(email: string, message: { text: string; blocks: object[] }): Promise<void>` -- Calls `lookupSlackUser(email)` first. If null, returns early. Otherwise calls `client.chat.postMessage({ channel: userId, text: message.text, blocks: message.blocks })`. Wraps the postMessage in try/catch, logs `console.error("[slack] Failed to DM ${email}:", error)` on failure (per D-16). Never throws.

Actions block with "View Shipment" URL button linking to `data.detailUrl`. Fallback text: `Shipment ${data.shipmentNumber} Delivered`.

b) `buildCompletedSlackMessage(data: SlackNotificationData)` -- Header: "Shipment {number} Completed". Section text: "All pieces have been picked up by the receiver." Section with 4 fields. Actions with "View Shipment" button. Fallback: `Shipment ${data.shipmentNumber} Completed`.

c) `buildInTransitSlackMessage(data: SlackNotificationData)` -- Header: "Shipment {number} In Transit". Section text: "This shipment is now on the way to its destination." Fields section. Actions with "View Shipment" button. Fallback: `Shipment ${data.shipmentNumber} In Transit`.

d) `buildAgedReminderSlackMessage(data: SlackNotificationData & { hoursAged: number })` -- Per D-07, includes urgency indicator. Compute `timeText`: if hoursAged >= 48 use `${Math.round(hoursAged / 24)} days`, else `${Math.round(hoursAged)} hours`. Header: "Pickup Reminder". Section text: `Shipment *${number}* was delivered *${timeText} ago* and is still awaiting pickup.` Fields: Sender, Pieces. Actions with "View Shipment" button. Fallback: `Reminder: Shipment ${number} awaiting pickup (${timeText})`.

e) `buildSignatureRequestSlackMessage(data: { shipmentNumber: string; signUrl: string; receiverName: string })` -- Per D-06, prominent "Sign Now" button. Header: "Signature Requested". Section text: `Shipment *${number}* has been delivered to *${receiverName}*. Please sign to confirm receipt.` Actions with "Sign Now" button using `style: "primary"` and `url: data.signUrl`, `action_id: "sign_shipment"`. Fallback: `Signature requested for shipment ${number}`.
